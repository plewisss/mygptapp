# CABSI NON SPECIALIZED WHOLESALE TRADING

Purchase order and receivables tracking app for CABSI.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/plewisss/mygptapp)

## Run Locally

```bash
npm install
npm start
```

## Deploy

The app is prepared for Render Blueprints. Click the button above, review the `cabsi-app` web service and `cabsi-db` Postgres database, then set `APP_USERNAME` and `APP_PASSWORD` in the Render service environment.
